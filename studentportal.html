Path@Penn Student Portal Redesign

Redesigning UPenn's student portal to create a more navigable experience that streamlines course registration and addresses key student pain points.

The Challenge

  • Students struggled with confusing navigation and couldn't easily find key portal features
  • Course information lacked transparency about difficulty, workload, and requirements
  • Permission request process was cumbersome with multiple unclear steps

The Goals

  • Improve navigation with intuitive information architecture and minimal clicks
  • Enhance course details with difficulty ratings, workload estimates, and reviews
  • Streamline permission request workflow with clear steps and status tracking

The Solution

  • Clean, consistent layout with balanced scaling and friendly typography
  • Comprehensive course information displayed in an accessible format
  • Simplified registration and permission request workflows
